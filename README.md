# Todo List CLI

A command line application to manage your daily tasks built with Node.js.

## Features
- Add a new task
- View all tasks
- Mark a task as complete
- Delete a task by ID

## Tech Stack
- Node.js
- fs module

## How to Run
node index.js add "Task name"
node index.js view
node index.js complete <id>
node index.js delete <id>
